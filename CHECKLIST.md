# Checklist - Контрольный список для массового скачивания RP5

## 📋 Перед началом работы

### Установка и настройка
- [ ] Python 3.8+ установлен
- [ ] Установлены зависимости: `pip install -r requirements.txt`
- [ ] Создана директория `data/rp5-csv/`
- [ ] Проверена работа Python: `python --version`

### Подготовка данных
- [ ] Подготовлен список WMO ID станций
- [ ] Определен период скачивания (start_date, end_date)
- [ ] Проверено наличие `data/wmo-mapping.js` (если нужен mapping)

## 🧪 Тестирование

### Шаг 1: Тест методов генерации
- [ ] Запущен: `python test-rp5-generation.py`
- [ ] Определен рабочий метод (1, 2, 3 или 4)
- [ ] Записаны результаты тестов

### Шаг 2: Тест на одной станции
- [ ] Протестирован Санкт-Петербург (26063)
- [ ] Файл успешно скачан
- [ ] Файл успешно распакован
- [ ] DataFrame загружен без ошибок

### Шаг 3: Анализ через DevTools (если тесты не прошли)
- [ ] Открыт https://rp5.ru/archive.php?wmo_id=26063
- [ ] Открыт DevTools (F12) → Network → XHR
- [ ] Нажата кнопка "Скачать архив"
- [ ] Найден POST запрос к `reFileSynop*.php`
- [ ] Скопированы параметры из Payload
- [ ] Обновлен `rp5_mass_downloader.py` с правильными параметрами

## 🚀 Запуск массового скачивания

### Выбор стратегии
- [ ] Выбрана стратегия:
  - [ ] Автоматическая (если POST работает)
  - [ ] Полуавтоматическая (рекомендуется)
  - [ ] Selenium (для полной автоматизации)

### Настройка параметров
- [ ] Задержки настроены (1-3 сек между станциями, 2-5 сек после генерации)
- [ ] Параллельность настроена (рекомендуется: max_workers=2)
- [ ] Retry логика настроена (рекомендуется: max_retries=5)

### Логирование
- [ ] Настроено логирование в файл
- [ ] Настроено логирование в консоль
- [ ] Определен уровень логирования (INFO или DEBUG)

### Запуск
- [ ] Создан скрипт для запуска
- [ ] Протестирован на 2-3 станциях
- [ ] Проверены результаты
- [ ] Запущено массовое скачивание

## 📊 Мониторинг

### Во время скачивания
- [ ] Мониторится лог файл
- [ ] Проверяется статистика (success/failed/retries)
- [ ] Проверяется использование диска
- [ ] Проверяется скорость скачивания

### После скачивания
- [ ] Проверено количество скачанных файлов
- [ ] Проверен размер файлов
- [ ] Проверена целостность CSV файлов
- [ ] Загружены DataFrame для проверки

## 🐛 Обработка ошибок

### Если все станции возвращают 403
- [ ] Переключиться на полуавтоматический режим
- [ ] Генерировать файлы вручную через браузер
- [ ] Скачивать автоматически через Python

### Если Error #FS001-0
- [ ] Проверить параметры POST через DevTools
- [ ] Рассмотреть использование Selenium
- [ ] Использовать полуавтоматический режим

### Если Timeout errors
- [ ] Увеличить timeout в запросах (до 60 сек)
- [ ] Проверить интернет-соединение
- [ ] Попробовать другой сервер (ru1, ru2, ru3)

### Если слишком медленно
- [ ] Увеличить параллельность (осторожно!)
- [ ] Уменьшить задержки (осторожно!)
- [ ] Разбить на партии и скачивать по расписанию
