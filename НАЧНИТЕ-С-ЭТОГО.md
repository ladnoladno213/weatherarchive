## 🚀 Быстрый старт (3 минуты)

### Шаг 1: Проверка

```bash
python check-setup.py
```

Должно быть: **"ВСЁ ГОТОВО К ЗАПУСКУ!"** ✅

### Шаг 2: Коммит

```bash
git add .
git commit -m "Add RP5 auto-update: 8 times daily, all cities, free"
git push
```

### Шаг 3: Запуск

1. Откройте GitHub → ваш репозиторий
2. Вкладка **Actions**
3. Найдите "RP5 Frequent Updates"
4. Нажмите **"Run workflow"** → **"Run workflow"**

### Шаг 4: Настройка прав

⚠️ **ВАЖНО!** После первого push:

1. GitHub → Settings → Actions → General
2. Scroll down to "Workflow permissions"
3. Выберите **"Read and write permissions"**
4. Save

Без этого workflow не сможет коммитить файлы!

## 🐛 Проблемы?

### Workflow не запускается

**Решение:** Settings → Actions → General → Workflow permissions → "Read and write permissions"

### Файлы не коммитятся

**Проверьте** что в `.gitignore` нет строки `rp5-realtime`

### Долго выполняется

**Нормально!** Первый запуск может занять до 10 минут.

Подробнее: **ЗАПУСК-GITHUB-ACTIONS.md** → раздел "Troubleshooting"

## 🎓 Дополнительно

### Изменить частоту обновлений

Редактируйте `.github/workflows/rp5-frequent.yml`:

```yaml
schedule:
  # Например, только 4 раза в день:
  - cron: '0 1 * * *'   # 06:00 Екатеринбург
  - cron: '0 7 * * *'   # 12:00 Екатеринбург
  - cron: '0 13 * * *'  # 18:00 Екатеринбург
  - cron: '0 19 * * *'  # 00:00 Екатеринбург
```

### Изменить период данных

Редактируйте `rp5-realtime-updater.py`:

```python
def get_date_range():
    end_date = datetime.now()
    start_date = end_date - timedelta(days=30)  # Изменить на 30 дней
    ...
```

### Добавить/удалить города

Редактируйте `data/wmo-mapping.js`:

```javascript
const wmoMapping = {
  524901: '28440',  // Москва
  // Добавьте свои города...
};
```
