# Автоматическая загрузка CSV с RP5

## Описание

Система автоматически скачивает исторические данные с сайта RP5.ru в формате CSV и использует их для отображения архива погоды.

## Как это работает

1. При запросе архива погоды система проверяет наличие CSV файла для города
2. Если файла нет или он устарел (> 24 часов), автоматически скачивается новый
3. CSV файлы сохраняются в папку `data/rp5-csv/`
4. Данные из CSV используются для T, Pa, U, Ff, ff10, Tn, Tx, VV, Td, RRR, tR, sss
5. WW/W1/W2 берутся из нашей логики (с мультимодельными грозами)

## Файлы

- `rp5-csv-downloader.js` - модуль для автоматической загрузки CSV
- `rp5-csv-loader.js` - модуль для парсинга CSV файлов
- `data/rp5-csv/` - папка для хранения скачанных CSV файлов

## Формат имени файла

```
{WMO_ID}.{start_date}.{end_date}.{city_name}.utf8.csv
```

Пример: `28573.2024-01-01.2024-12-31.ishim.utf8.csv`

## Добавление новых городов

Отредактируйте `rp5-csv-downloader.js`:

```javascript
const GEONAME_TO_WMO = {
  '1505453': '28573', // Ишим
  '524901': '27612',  // Москва (пример)
};

const GEONAME_TO_CITY = {
  '1505453': 'ishim',
  '524901': 'moscow',
};
```

## Параметры загрузки

- Кодировка: UTF-8 (f_ed3=8)
- Разделитель: точка с запятой (f_ed4=1)
- Формат: только синоптические сроки (f_ed5=1)
- Дни: все дни (f_ed2=1)

## Кэширование

- CSV файлы кэшируются на 24 часа
- После этого автоматически скачивается обновлённая версия
- Можно вручную удалить файл из `data/rp5-csv/` для принудительного обновления

## Тестирование

Запустите тестовый скрипт:

```bash
node test-rp5-downloader.js
```

Это скачает CSV для Ишима за последний месяц и покажет путь к файлу.

## Источник данных

Данные предоставлены сайтом "Расписание Погоды" (rp5.ru).
При использовании данных необходимо указывать источник.

## Преимущества

1. Реальные наблюдения вместо модельных данных
2. Более точные значения температуры, давления, влажности
3. Автоматическое обновление данных
4. Кэширование для быстрой загрузки
5. Комбинация реальных данных с улучшенной логикой гроз

## Ограничения

- Доступны только города с метеостанциями WMO
- Требуется интернет-соединение для первой загрузки
- RP5 может ограничивать частоту запросов
