# ЗАПУСК - Простая инструкция

## Шаг 1: Проверка готовности

```powershell
python check-ready.py
```

Этот скрипт проверит:
- Установлен ли Selenium
- Установлен ли Requests
- Есть ли ChromeDriver
- Есть ли папка для файлов

## Шаг 2: Если что-то не готово

### Если нет Selenium или Requests:

```powershell
pip install selenium requests
```

### Если нет ChromeDriver:

У вас уже есть! Папка `chromedriver-win64` с файлом `chromedriver.exe`

## Шаг 3: Запуск теста

```powershell
python download-one-station.py
```

Что произойдет:
1. Откроется браузер Chrome
2. Автоматически откроется сайт RP5
3. Заполнятся даты (01.01.2024 - 31.12.2024)
4. Нажмется кнопка "Скачать архив"
5. Подождет 15 секунд
6. Скачает файл
7. Распакует его
8. Закроет браузер

Файл появится: `data/rp5-csv/26063.csv`

## Шаг 4: Массовое скачивание

Если тест прошел успешно:

```powershell
python download-mass.py
```

Скачает 5 станций:
- Санкт-Петербург (26063)
- Москва (27612)
- Ишим (28573)
- Новосибирск (29634)
- Екатеринбург (28698)
