# FAQ - Простые ответы на частые вопросы

## ❓ Общие вопросы

### Что делает этот скрипт?

Автоматически открывает браузер, заходит на сайт RP5, заполняет форму, нажимает кнопку "Скачать архив", ждет генерации файла и скачивает его.

### Нужно ли мне что-то делать вручную?

Нет! Просто запустите скрипт и наблюдайте. Всё происходит автоматически.

### Сколько времени это занимает?

- Одна станция: ~20-30 секунд
- 5 станций: ~2-3 минуты
- 10 станций: ~5-6 минут

### Можно ли скачать сразу 100 станций?

Да, но лучше разбить на партии по 10-20 станций, чтобы не перегружать сайт.

## 🔧 Технические вопросы

### Где находится ChromeDriver?

В папке `chrome-win64` в корне проекта. Там должен быть файл `chromedriver.exe`.

### Как узнать WMO ID станции?

Откройте файл `data/wmo-mapping.js` - там есть mapping всех городов на станции.

Или зайдите на https://rp5.ru/ и найдите нужный город - в URL будет `wmo_id=XXXXX`.

### Можно ли запустить без отображения браузера?

Да! В файле `download-mass.py` измените `HEADLESS = True`.

### Как изменить период скачивания?

В файлах `download-one-station.py` или `download-mass.py` измените:
```python
START_DATE = '01.01.2020'  # Ваша дата
END_DATE = '31.12.2024'    # Ваша дата
```

## 🐛 Проблемы и решения

### "ChromeDriver не найден"

**Причина:** ChromeDriver не скачан или находится не в той папке

**Решение:** 
```bash
python download-chromedriver.py
```

Это автоматически скачает ChromeDriver в правильную папку.

**Или вручную:**
1. Откройте https://googlechromelabs.github.io/chrome-for-testing/
2. Найдите "Stable" → "chromedriver" → "win64"
3. Скачайте zip файл
4. Распакуйте в папку проекта
5. Должна получиться папка: `chromedriver-win64` с файлом `chromedriver.exe`

### "selenium module not found"

**Причина:** Selenium не установлен

**Решение:**
```bash
pip install selenium
```

### Браузер открывается и сразу закрывается

**Причина:** Произошла ошибка в скрипте

**Решение:** Посмотрите в консоль - там будет сообщение об ошибке. Скопируйте его и изучите.

### Кнопка "Скачать архив" не найдена

**Причина:** Сайт изменил структуру или страница не загрузилась

**Решение:** Увеличьте время ожидания:
```python
time.sleep(3)  # Измените на 5 или 10
```

### Файл не скачивается (403 Forbidden)

**Причина:** Файл еще не сгенерировался на сервере

**Решение:** Увеличьте время ожидания после нажатия кнопки:
```python
time.sleep(15)  # Измените на 30 или даже 60
```

### Скрипт зависает

**Причина:** Долгая генерация файла или проблемы с интернетом

**Решение:** 
1. Подождите еще немного
2. Нажмите Ctrl+C для остановки
3. Запустите снова

## 📁 Файлы и папки

### Где скачанные файлы?

В папке `data/rp5-csv/`

### Какой формат файлов?

CSV (текстовый файл с разделителями точка с запятой)

### Можно ли открыть в Excel?

Да! Откройте Excel → Файл → Открыть → Выберите CSV файл

### Как проверить, что файл скачался правильно?

Откройте его в блокноте. Должны быть строки с данными погоды, разделенные точкой с запятой.

## 🎯 Использование

### Как скачать только одну станцию?

```bash
python download-one-station.py
```

### Как скачать несколько станций?

```bash
python download-mass.py
```

### Как добавить свои станции?

Откройте `download-mass.py` в блокноте и измените список:
```python
STATIONS = [
    '26063',  # Ваши станции
    '27612',
    '12345',  # Добавьте сколько нужно
]
```

### Как изменить период?

В том же файле измените:
```python
START_DATE = '01.01.2020'
END_DATE = '31.12.2024'
```

## ⚙️ Настройки

### Как ускорить скачивание?

1. Уменьшите задержку между станциями (осторожно!):
```python
time.sleep(3)  # Измените на 1 или 2
```

2. Включите headless режим:
```python
HEADLESS = True
```

### Как замедлить (если сайт блокирует)?

Увеличьте задержки:
```python
time.sleep(3)  # Измените на 5 или 10
```

### Как скрыть браузер?

```python
HEADLESS = True
```

### Как показать браузер (для отладки)?

```python
HEADLESS = False
```

## 📊 Статистика

### Как узнать, сколько файлов скачано?

Посмотрите в консоль - в конце будет итоговая статистика:
