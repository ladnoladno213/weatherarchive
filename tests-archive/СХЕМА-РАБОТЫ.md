## 🔄 Детальный процесс скачивания

```
Для каждой станции из data/wmo-mapping.js:

1. Открыть страницу
   https://rp5.ru/archive.php?wmo_id=XXXXX
   
2. Кликнуть на вкладку
   "Скачать архив погоды"
   
3. Заполнить даты
   ┌─────────────────────────┐
   │ С: 26.03.2026          │
   │ По: 02.04.2026         │
   └─────────────────────────┘
   
4. Выбрать формат
   ○ HTML
   ● CSV  ← выбрать
   ○ XML
   
5. Выбрать кодировку
   ○ Windows-1251
   ● UTF-8  ← выбрать
   
6. Нажать кнопку
   [Выбрать в файл GZ (архив)]
   
7. Подождать 5-7 секунд
   (генерация архива на сервере)
   
8. Кликнуть на ссылку
   "Скачать" (появляется справа)
   
9. Скачать .gz файл
   
10. Распаковать в .csv
    
11. Сохранить в
    data/rp5-realtime/XXXXX.csv
    
12. Пауза 10 секунд
    (чтобы не перегружать RP5)
    
13. Следующая станция →
```

## 📄 Формат CSV файла

```
Заголовок (6 строк):
┌────────────────────────────────────────────────────┐
│ Местное время в Москве (ВДНХ)                     │
│ Сведения получены с сайта: https://rp5.ru         │
│ Период: 26.03.2026 - 02.04.2026                   │
│ Формат: CSV (разделитель ;)                       │
│ Кодировка: UTF-8                                   │
│                                                    │
└────────────────────────────────────────────────────┘

Данные (каждая строка = 1 наблюдение):
┌────────────────────────────────────────────────────┐
│ Местное время;T;Po;P;Pa;U;DD;Ff;ff10;ff3;N;WW;... │
│ 02.04.2026 00:00;-2.1;748;747;-0.3;89;ЮЗ;3;5;7... │
│ 02.04.2026 03:00;-2.5;749;748;-0.2;91;ЮЗ;2;4;6... │
│ 02.04.2026 06:00;-3.0;750;749;-0.1;93;Ю;1;3;5;... │
│ ...                                                │
└────────────────────────────────────────────────────┘

Поля:
• T    - температура (°C)
• Po   - давление на уровне станции (мм рт.ст.)
• P    - давление на уровне моря (мм рт.ст.)
• U    - влажность (%)
• DD   - направление ветра
• Ff   - скорость ветра (м/с)
• WW   - текущая погода (текст)
• W1   - прошедшая погода 1
• W2   - прошедшая погода 2
• ...  - и другие параметры
```
