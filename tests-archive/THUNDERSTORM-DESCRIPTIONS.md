# Описания гроз в архиве погоды

## Обзор

Система предоставляет детальные описания гроз на основе кодов погоды WMO и метеорологических параметров.

## Типы описаний

### 1. Гроза слабая
**Условия:**
- Код погоды: 95
- CAPE ≤ 1000 J/kg
- Осадки ≤ 10 мм

**Примеры:**
- "Гроза слабая"
- "Гроза слабая, но без осадков" (если осадки < 0.1 мм)

### 2. Гроза умеренная
**Условия:**
- Код погоды: 95
- CAPE > 1000 J/kg ИЛИ осадки > 10 мм

**Примеры:**
- "Гроза умеренная"
- "Гроза умеренная, но без осадков" (если осадки < 0.1 мм)

### 3. Гроза с небольшим градом
**Условия:**
- Код погоды: 96

**Описание:**
- "Гроза с небольшим градом"

### 4. Гроза сильная
**Условия:**
- Код погоды: 97

**Примеры:**
- "Гроза сильная"
- "Гроза сильная, но без осадков" (если осадки < 0.1 мм)

### 5. Гроза с пыльной бурей
**Условия:**
- Код погоды: 98

**Описание:**
- "Гроза с пыльной бурей"

### 6. Гроза с сильным градом
**Условия:**
- Код погоды: 99

**Описание:**
- "Гроза с сильным градом"

### 7. Гроза в течение последнего часа
**Условия:**
- Гроза была обнаружена в предыдущем часе
- В текущем часе грозы нет

**Примеры:**
- "Гроза в течение последнего часа"
- "Гроза в течение последнего часа, дождь" (если есть другие явления)

## Особенности описаний

### Осадки
Если осадки < 0.1 мм, добавляется уточнение:
```
", но без осадков"
```

**Примеры:**
- "Гроза слабая, но без осадков"
- "Гроза сильная, но без осадков"

### Комбинация с другими явлениями
Если есть другие погодные явления, они добавляются после описания грозы:
```
"Гроза умеренная, дождь"
"Гроза слабая, туман"
```

### Ясная погода
При ясной погоде (облачность < 30%) грозы обычно не бывают, но если система их обнаруживает (например, по высокому CAPE), описание формируется стандартно.

## Логика формирования описания

```javascript
function getThunderstormDescription(code, cape, totalPrecip) {
  const hasPrecip = totalPrecip > 0.1;
  
  if (code === 95) {
    let description = (cape > 1000 || totalPrecip > 10) 
      ? 'Гроза умеренная' 
      : 'Гроза слабая';
    
    if (!hasPrecip) {
      description += ', но без осадков';
    }
    
    return description;
  }
  
  // ... другие коды
}
```

Результат: "Гроза умеренная"
```

```

Код: 95

```

### Пример 3: Гроза прошла
```

## Интеграция с мультимодельной системой

Описание грозы формируется на основе:
1. **Максимального кода погоды** среди всех моделей
2. **Максимального CAPE** среди всех моделей
3. **Максимальных осадков** среди всех моделей

Это обеспечивает наиболее полное описание грозовой активности.

## Уровни уверенности

Описание грозы сопровождается уровнем уверенности:
- **Очень высокая**: 3+ моделей обнаружили грозу
- **Высокая**: 2 модели обнаружили грозу
- **Средняя**: 1 модель обнаружила грозу

Уровень уверенности не отображается пользователю, но используется для логирования и отладки.
