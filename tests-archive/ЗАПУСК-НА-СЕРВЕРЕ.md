# Запуск автоматического скачивания на сервере

## Что получилось

✅ **Рабочий скрипт**: `rp5-auto-downloader.py`
- Автоматически открывает страницу RP5
- Заполняет даты
- Выбирает CSV и UTF-8
- Нажимает кнопки
- Скачивает файл в указанную папку

## Локальный запуск (Windows)

```bash
# С GUI (видно браузер)
py rp5-auto-downloader.py

# Без GUI (headless)
# Отредактируйте в скрипте: HEADLESS = True
py rp5-auto-downloader.py
```

Файлы сохраняются в папку `data/rp5-csv/`

## Запуск на сервере (Linux)

### 1. Установка зависимостей

```bash
# Установка Chrome
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb
sudo apt-get install -f

# Установка ChromeDriver
wget https://storage.googleapis.com/chrome-for-testing-public/147.0.7727.24/linux64/chromedriver-linux64.zip
unzip chromedriver-linux64.zip
sudo mv chromedriver-linux64/chromedriver /usr/local/bin/
sudo chmod +x /usr/local/bin/chromedriver

# Установка Python пакетов
pip install selenium requests
```

### 2. Настройка скрипта для Linux

Отредактируйте `rp5-auto-downloader.py`:

```python
# Измените пути для Linux
chrome_paths = [
    '/usr/bin/google-chrome',  # Системный Chrome
]

driver_paths = [
    '/usr/local/bin/chromedriver',  # Системный ChromeDriver
]
```

### 3. Автоматический запуск (cron)

```bash
# Открыть crontab
crontab -e

# Добавить задачу (запуск каждый день в 3:00 ночи)
0 3 * * * cd /path/to/project && python3 rp5-auto-downloader.py >> logs/rp5-download.log 2>&1
```

## Настройка параметров

Отредактируйте в `rp5-auto-downloader.py`:

```python
# Станция
STATION_ID = '28573'  # Ваш WMO ID

# Период
START_DATE = '01.01.2016'
END_DATE = '31.03.2026'

# Папка для сохранения
OUTPUT_DIR = 'data/rp5-csv'

# Режим работы
HEADLESS = True  # True для сервера, False для отладки
```

## Массовое скачивание

Для скачивания нескольких станций создайте список:

```python
stations = [
    ('28573', '01.01.2016', '31.03.2026'),  # Ишим
    ('27612', '01.01.2016', '31.03.2026'),  # Москва
    ('26063', '01.01.2016', '31.03.2026'),  # Санкт-Петербург
]

for station_id, start, end in stations:
    file_path = downloader.download_station(station_id, start, end)
    time.sleep(60)  # Пауза между станциями
```

## Проблемы и решения

### Ошибка: Chrome не найден
```bash
# Проверьте установку
which google-chrome
which chromedriver
```

### Ошибка: Permission denied
```bash
# Дайте права на выполнение
chmod +x rp5-auto-downloader.py
```

### Файлы скачиваются не туда
- Проверьте параметр `OUTPUT_DIR` в скрипте
- Убедитесь что папка существует и есть права на запись

## Альтернатива: Docker

Можно упаковать все в Docker контейнер для простого развертывания на любом сервере.
