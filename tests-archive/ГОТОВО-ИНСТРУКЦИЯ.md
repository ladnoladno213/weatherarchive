# ✅ ГОТОВО! Автоматическое скачивание работает

## Что сделано

1. ✅ Создан рабочий скрипт `rp5-auto-downloader.py`
2. ✅ Скрипт автоматически:
   - Открывает страницу RP5
   - Заполняет даты
   - Выбирает CSV и UTF-8
   - Нажимает все нужные кнопки
   - Скачивает файл в папку `data/rp5-csv/`

## Как использовать

### Простой запуск (один файл)

```bash
py rp5-auto-downloader.py
```

Файл сохранится в `data/rp5-csv/28573.csv`

### Изменить параметры

Откройте `rp5-auto-downloader.py` и измените:

```python
STATION_ID = '28573'      # Ваша станция
START_DATE = '01.01.2016' # Начало периода
END_DATE = '31.03.2026'   # Конец периода
OUTPUT_DIR = 'data/rp5-csv'  # Папка для файлов
HEADLESS = True           # False чтобы видеть браузер
```

### Скачать несколько станций

Добавьте в конец `main()`:

```python
stations = [
    ('28573', '01.01.2016', '31.03.2026'),
    ('27612', '01.01.2016', '31.03.2026'),
    ('26063', '01.01.2016', '31.03.2026'),
]

for station_id, start, end in stations:
    logger.info(f"\nОбработка станции {station_id}...")
    file_path = downloader.download_station(station_id, start, end)
    if file_path:
        logger.info(f"✓ Готово: {file_path}")
    time.sleep(60)  # Пауза между станциями
```
