# Финальные критерии обнаружения гроз

## Обзор системы

Мультимодельная система использует **11 глобальных метеорологических моделей** для обнаружения гроз в архивных данных.

## Список моделей

1. **JMA GSM** (Япония, 55 км) - с 2016-01-01
2. **ECMWF IFS HRES** (Европа, 9 км) - с 2017-01-01
3. **GFS Global** (США, 13 км) - с 2021-03-23
4. **UKMO Global** (Великобритания, 10 км) - с 2022-03-01
5. **ECMWF IFS 0.4°** (Европа, 44 км) - с 2022-11-07
6. **GEM Global** (Канада, 15 км) - с 2022-11-23
7. **ICON Global** (Германия, 11 км) - с 2022-11-24
8. **CMA GRAPES** (Китай, 15 км) - с 2023-12-31
9. **ARPEGE World** (Франция, 25 км) - с 2024-01-02
10. **BOM ACCESS** (Австралия, 15 км) - с 2024-01-18
11. **ECMWF IFS 0.25°** (Европа, 25 км) - с 2024-02-03

## Критерии обнаружения

### Метод 1: Прямое обнаружение по коду погоды
```
weather_code >= 95 AND weather_code <= 99
```
**Коды:**
- 95 = Гроза слабая/умеренная
- 96 = Гроза с небольшим градом
- 97 = Гроза сильная
- 98 = Гроза с пыльной бурей
- 99 = Гроза с сильным градом

### Метод 2: Ливни + конвекция
```
weather_code >= 80 AND weather_code <= 82 AND CAPE > 500 J/kg
```
Ливневые осадки (коды 80-82) в сочетании с умеренной конвекцией указывают на грозу.

### Метод 3: Дождь + конвекция
```
weather_code >= 51 AND weather_code <= 67 AND CAPE > 800 J/kg AND precipitation > 0.3 mm
```
Обычные осадки (коды 51-67) с высокой конвекцией и осадками.

### Метод 4: Умеренная конвекция + осадки
```
CAPE > 500 J/kg AND precipitation > 2 mm
```

### Метод 5: Высокая конвекция + небольшие осадки
```
CAPE > 1000 J/kg AND precipitation > 0.3 mm
```

### Метод 6: Очень высокая конвекция
```
CAPE > 1500 J/kg AND precipitation > 0.1 mm
```

### Метод 7: Экстремальная конвекция
```
CAPE > 2000 J/kg
```
Даже без осадков, экстремальная конвективная энергия указывает на грозовую активность.

### Метод 8: Очень сильные осадки
```
precipitation > 10 mm/h
```
Интенсивные осадки часто связаны с грозами.

## Уровни уверенности

Уверенность определяется по количеству моделей, обнаруживших грозу:

- **Очень высокая** (very_high): 3+ моделей из 11 (≥27%)
- **Высокая** (high): 2 модели из 11 (≥18%)
- **Средняя** (medium): 1 модель из 11 (≥9%)

## Описание интенсивности грозы

Для кода 95 (гроза слабая/умеренная) уточняется интенсивность:

```javascript
if (code === 95) {
  if (CAPE > 1000 OR precipitation > 10mm) {
    return "Гроза умеренная"
  } else {
    return "Гроза слабая"
  }
}
```

## Преимущества подхода

1. **Высокая надёжность** - использование 11 моделей снижает ложные срабатывания
2. **Глубокая история** - данные с 2016 года благодаря JMA GSM
3. **Гибкие критерии** - 8 различных методов обнаружения
4. **Глобальное покрытие** - работает для любой точки мира
5. **Устойчивость** - система продолжает работать даже если часть моделей недоступна

## Ограничения

1. **Модельные данные** - не всегда совпадают с реальными наблюдениями
2. **Разрешение** - глобальные модели (9-55 км) могут пропускать локальные грозы
3. **Доступность** - не все модели доступны для всех периодов
4. **Консервативность** - система может пропускать слабые грозы

## Рекомендации по использованию

### Для критических приложений:
- Учитывайте уровень уверенности
- Очень высокая/Высокая - надёжные данные
- Средняя - возможны пропуски

### Для исторического анализа:
- 2024-2025: все 11 моделей доступны (максимальная точность)
- 2022-2023: 7-10 моделей доступны (высокая точность)
- 2021: 3 модели доступны (средняя точность)
- 2017-2020: 2 модели доступны (базовая точность)
- 2016: 1 модель доступна (минимальная точность)

### Для повышения точности:
- Используйте данные за последние 2 года (2024-2025)
- Обращайте внимание на количество согласных моделей
- Учитывайте региональные особенности климата
