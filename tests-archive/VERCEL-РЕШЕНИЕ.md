# Решение для Vercel: GitHub Actions

## Проблема

Vercel - это serverless платформа, там нельзя:
- ❌ Запускать Selenium
- ❌ Устанавливать Chrome
- ❌ Выполнять долгие задачи (timeout 10 сек)

## Решение: GitHub Actions

✅ **GitHub Actions** - бесплатный CI/CD от GitHub
- Скачивает файлы автоматически
- Сохраняет в репозиторий
- Vercel деплоит обновленные файлы

## Как это работает

```
GitHub Actions (раз в месяц)
    ↓
Запускает rp5-github-action.py
    ↓
Скачивает CSV файлы
    ↓
Коммитит в репозиторий
    ↓
Vercel автоматически деплоит
    ↓
Сайт использует свежие данные
```

## Настройка

### 1. Файлы уже созданы

- `.github/workflows/download-rp5.yml` - конфигурация GitHub Action
- `rp5-github-action.py` - скрипт скачивания

### 2. Добавьте станции

Отредактируйте `rp5-github-action.py`:

```python
stations = [
    ('28573', '01.01.2016', '31.03.2026'),  # Ишим
    ('27612', '01.01.2016', '31.03.2026'),  # Москва
    ('26063', '01.01.2016', '31.03.2026'),  # Санкт-Петербург
]
```

### 3. Закоммитьте в GitHub

```bash
git add .github/workflows/download-rp5.yml
git add rp5-github-action.py
git commit -m "Add GitHub Actions for RP5 download"
git push
```

### 4. Настройте расписание

В файле `.github/workflows/download-rp5.yml`:

```yaml
schedule:
  - cron: '0 3 1 * *'  # Каждый месяц 1-го числа в 3:00 UTC
```

Примеры:
- `0 3 * * *` - каждый день в 3:00
- `0 3 * * 0` - каждое воскресенье в 3:00
- `0 3 1,15 * *` - 1-го и 15-го числа в 3:00

### 5. Ручной запуск

1. Откройте GitHub → ваш репозиторий
2. Вкладка "Actions"
3. Выберите "Download RP5 Weather Data"
4. Нажмите "Run workflow"

## Преимущества

✅ **Бесплатно** - 2000 минут в месяц
✅ **Автоматически** - работает по расписанию
✅ **Надежно** - GitHub инфраструктура
✅ **Vercel совместимо** - файлы в репозитории
✅ **Без сервера** - не нужен VPS

## Ограничения

- ⏱️ Timeout 6 часов на задачу
- 📦 Размер файлов до 100 MB
- 🔄 2000 минут в месяц (бесплатно)

Для ваших задач этого более чем достаточно!

## Альтернативы

### Вариант 2: Vercel Cron + External API

Создать отдельный сервис на Railway/Render, Vercel вызывает его по cron.

### Вариант 3: Cloudflare Workers

Использовать Cloudflare Workers + Puppeteer для скачивания.

### Вариант 4: Ручное обновление

Запускать `rp5-auto-downloader.py` локально раз в месяц.

## Рекомендация

Используйте **GitHub Actions** - это самое простое и надежное решение для Vercel!

## Проверка работы

После первого запуска:

1. Проверьте Actions → последний запуск
2. Убедитесь что файлы появились в `data/rp5-csv/`
3. Vercel автоматически задеплоит изменения
4. Ваш сайт будет использовать свежие данные

## Мониторинг

GitHub отправит email если Action упадет с ошибкой.
