## Источники с РЕАЛЬНЫМИ наблюдениями

### ⭐ 1. Meteostat API (РЕКОМЕНДУЕТСЯ)
**URL**: https://dev.meteostat.net/api  
**Платформа**: RapidAPI

**Преимущества**:
- ✅ Реальные наблюдения с метеостанций
- ✅ Данные от правительственных источников
- ✅ Почасовые данные
- ✅ Бесплатный тариф: 500 запросов/месяц
- ✅ Максимум 30 дней за запрос
- ✅ Любые координаты (автоматически находит ближайшие станции)

**Недостатки**:
- ❌ Требует регистрацию на RapidAPI
- ❌ Ограничение: 500 запросов/месяц на бесплатном тарифе

**Регистрация**:
1. Зарегистрироваться на https://rapidapi.com
2. Подписаться на Meteostat API: https://rapidapi.com/meteostat/api/meteostat
3. Выбрать бесплатный план (Basic - 500 calls/month)
4. Получить API ключ (X-RapidAPI-Key)

**Пример запроса**:
```bash
curl --request GET \
  --url 'https://meteostat.p.rapidapi.com/point/hourly?lat=55.7558&lon=37.6173&start=2026-03-25&end=2026-04-01&tz=Europe/Moscow' \
  --header 'x-rapidapi-host: meteostat.p.rapidapi.com' \
  --header 'x-rapidapi-key: YOUR_KEY_HERE'
```

**Формат ответа**:
```json
{
  "meta": {
    "generated": "2026-04-01 12:00:00"
  },
  "data": [
    {
      "time": "2026-03-25 00:00:00",
      "temp": 5.2,
      "dwpt": 2.1,
      "rhum": 85,
      "prcp": 0.0,
      "snow": 0,
      "wdir": 180,
      "wspd": 12,
      "wpgt": 18,
      "pres": 1013.2,
      "tsun": 0,
      "coco": 4
    }
  ]
}
```

**Параметры данных**:
- `time` - время наблюдения (YYYY-MM-DD HH:MM:SS)
- `temp` - температура воздуха (°C)
- `dwpt` - точка росы (°C)
- `rhum` - относительная влажность (%)
- `prcp` - осадки за час (мм)
- `snow` - высота снежного покрова (мм)
- `wdir` - направление ветра (градусы)
- `wspd` - скорость ветра (км/ч)
- `wpgt` - порывы ветра (км/ч)
- `pres` - давление на уровне моря (гПа)
- `tsun` - солнечное сияние за час (минуты)
- `coco` - код погодных условий

**Коды погодных условий (coco)**:
- 1 = Clear (ясно)
- 2 = Fair (малооблачно)
- 3 = Cloudy (облачно)
- 4 = Overcast (пасмурно)
- 5 = Fog (туман)
- 6 = Freezing Fog (ледяной туман)
- 7 = Light Rain (слабый дождь)
- 8 = Rain (дождь)
- 9 = Heavy Rain (сильный дождь)
- 10 = Freezing Rain (ледяной дождь)
- 11 = Heavy Freezing Rain (сильный ледяной дождь)
- 12 = Sleet (мокрый снег)
- 13 = Heavy Sleet (сильный мокрый снег)
- 14 = Light Snowfall (слабый снег)
- 15 = Snowfall (снег)
- 16 = Heavy Snowfall (сильный снег)
- 17 = Rain Shower (ливень)
- 18 = Heavy Rain Shower (сильный ливень)
- 19 = Sleet Shower (ливень с мокрым снегом)
- 20 = Heavy Sleet Shower (сильный ливень с мокрым снегом)
- 21 = Snow Shower (снежный ливень)
- 22 = Heavy Snow Shower (сильный снежный ливень)
- 23 = Lightning (молния)
- 24 = Hail (град)
- 25 = Thunderstorm (гроза)
- 26 = Heavy Thunderstorm (сильная гроза)
- 27 = Storm (шторм)

### 2. Aviation Weather API (aviationweather.gov)
**URL**: https://aviationweather.gov/api/data/metar

**Преимущества**:
- ✅ Полностью бесплатно
- ✅ НЕ требует регистрации или API ключа
- ✅ Данные METAR от метеостанций аэропортов
- ✅ Исторические данные до 15 дней
- ✅ Реальные наблюдения

**Недостатки**:
- ❌ Только аэропорты (не все города)
- ❌ Лимит: 100 запросов в минуту

**Статус**: Уже используется для страницы архива аэропортов

### 3. NOAA NCEI API
**URL**: https://www.ncei.noaa.gov/cdo-web/webservices/v2

**Преимущества**:
- ✅ Официальные данные США
- ✅ Реальные наблюдения
- ✅ Бесплатно

**Недостатки**:
- ❌ Требует регистрацию для токена
- ❌ Лимит: 10,000 запросов/день
- ❌ В основном станции США

## Рекомендация для архива городов

Использовать **Meteostat API** через RapidAPI:

1. Зарегистрироваться на RapidAPI и получить ключ
2. Добавить ключ в server.js (функция `fetchMeteostat`)
3. Использовать Meteostat для страницы `/archivetest`
4. Бесплатный лимит 500 запросов/месяц должен быть достаточен для тестирования

## Реализация

Функция `fetchMeteostat` уже добавлена в server.js. Нужно только:
1. Получить RapidAPI ключ
2. Заменить `YOUR_RAPIDAPI_KEY_HERE` на реальный ключ
3. Добавить парсинг данных Meteostat в маршрут `/archivetest`
