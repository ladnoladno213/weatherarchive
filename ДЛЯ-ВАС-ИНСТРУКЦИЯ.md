# Инструкция специально для вас

## Проблема решена! ✅

У вас скачался Chrome, а не ChromeDriver. Это разные вещи.

## Что делать (просто скопируйте команды)

### 1. Установить Selenium и requests

```powershell
pip install selenium requests
```

### 2. Скачать ChromeDriver автоматически

```powershell
python download-chromedriver.py
```

Этот скрипт:
- Автоматически найдет последнюю версию ChromeDriver
- Скачает её
- Распакует в правильную папку
- Всё настроит

### 3. Запустить тест

```powershell
python download-one-station.py
```

Откроется браузер и начнется скачивание.

### 4. Если тест прошел - запустить массовое скачивание

```powershell
python download-mass.py
```

## Что произойдет

После команды `python download-chromedriver.py`:

1. Скрипт подключится к серверу Google
2. Узнает последнюю версию ChromeDriver
3. Скачает zip файл (~10 МБ)
4. Распакует в папку `chromedriver-win64`
5. Покажет сообщение "SUCCESS"

После этого можно запускать `python download-one-station.py`

## Если download-chromedriver.py не работает

Скачайте вручную:

1. Откройте: https://googlechromelabs.github.io/chrome-for-testing/
2. Найдите раздел "Stable"
3. Найдите "chromedriver" → "win64"
4. Нажмите на ссылку (скачается zip файл)
5. Распакуйте zip в папку проекта
6. Должна получиться папка: `chromedriver-win64`
7. В ней должен быть файл: `chromedriver.exe`

## Проверка

После скачивания проверьте, что файл на месте:

```powershell
dir chromedriver-win64
```

Должен показать файл `chromedriver.exe`

## Готово!

Теперь запускайте:

```powershell
python download-one-station.py
```

И всё заработает! 🚀
