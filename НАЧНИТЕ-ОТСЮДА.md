# 🚀 НАЧНИТЕ ОТСЮДА - Самое простое руководство

## Что нужно сделать (4 шага)

### Шаг 1: Установить Selenium

Откройте PowerShell и выполните:

```bash
pip install selenium requests
```

### Шаг 2: Скачать ChromeDriver

```bash
python download-chromedriver.py
```

Это автоматически скачает и установит ChromeDriver.

### Шаг 3: Запустить тест

```bash
python download-one-station.py
```

### Шаг 4: Если тест прошел - запустить массовое скачивание

```bash
python download-mass.py
```

## Всё! Готово! 🎉

---

## Что произойдет?

1. Откроется браузер Chrome
2. Автоматически откроется сайт RP5
3. Заполнятся даты
4. Нажмется кнопка "Скачать архив"
5. Подождет 15 секунд
6. Скачает файл
7. Распакует его
8. Закроет браузер

Файлы сохранятся в папке: `data/rp5-csv/`

---

## Если что-то не работает

### Ошибка: "selenium module not found"

```bash
pip install selenium
```

### Ошибка: "ChromeDriver не найден"

Проверьте, что файл `chromedriver.exe` находится в папке `chrome-win64`

### Браузер открывается и сразу закрывается

Посмотрите в консоль - там будет сообщение об ошибке

### Файл не скачивается

Подождите дольше - увеличьте время ожидания в скрипте

## Как изменить настройки

### Изменить станцию

Откройте `download-one-station.py` в блокноте:

```python
STATION_ID = '26063'  # Измените на нужный WMO ID
```

### Изменить период

```python
START_DATE = '01.01.2024'  # Ваша дата
END_DATE = '31.12.2024'    # Ваша дата
```

### Добавить станции для массового скачивания

Откройте `download-mass.py` в блокноте:

```python
STATIONS = [
    '26063',  # Добавьте или удалите
    '27612',
    '28573',
]
```

## Быстрая проверка

### 1. Selenium установлен?

```bash
python -c "import selenium; print('OK')"
```

Должно вывести: `OK`

### 2. ChromeDriver на месте?

Откройте папку `chrome-win64` - там должен быть `chromedriver.exe`

### 3. Всё готово!

```bash
python download-one-station.py
```

## Что дальше?

После успешного теста:

1. Откройте `download-mass.py`
2. Добавьте нужные станции в список `STATIONS`
3. Измените период `START_DATE` и `END_DATE`
4. Запустите: `python download-mass.py`
5. Наблюдайте за процессом
6. Файлы появятся в `data/rp5-csv/`
