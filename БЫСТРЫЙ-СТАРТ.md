# 🚀 Быстрый старт - Автообновление RP5

## ✅ Всё готово к запуску!

Все файлы созданы и настроены. Осталось только запустить.

## 📋 Три простых шага

### 1️⃣ Закоммитьте и запушьте

```bash
git add .
git commit -m "Add RP5 auto-update every 3 hours"
git push
```

### 2️⃣ Включите GitHub Actions

1. Откройте https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ
2. Вкладка **Actions**
3. Если нужно - нажмите "Enable Actions"

### 3️⃣ Запустите первый раз

1. Actions → "RP5 Frequent Updates"
2. **Run workflow** → **Run workflow**
3. Ждите 5-10 минут
4. Готово! ✅

## 📊 Что получите

- ✅ Обновления каждые 3 часа (8 раз в день)
- ✅ Данные для всех городов с прогнозом

## ❓ Проблемы?

### Workflow не запускается

**Решение:** Settings → Actions → General → Workflow permissions → "Read and write permissions"

### Долго выполняется

**Нормально!** Первый запуск может занять до 10 минут (скачивает Chrome).

### Файлы не появляются

1. Проверьте логи: Actions → последний запуск → "update"
2. Убедитесь что нет ошибок
3. Сделайте `git pull`
