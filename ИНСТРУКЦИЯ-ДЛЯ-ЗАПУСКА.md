# Инструкция для запуска - Простое руководство

## 🎯 Что нужно сделать

### Шаг 1: Установить Selenium (если еще не установлен)

Откройте командную строку (PowerShell) и выполните:

```bash
pip install selenium
```

### Шаг 2: Проверить ChromeDriver

У вас уже есть папка `chrome-win64` с драйвером. Проверьте, что там есть файл `chromedriver.exe`.

Если его нет, скачайте отсюда: https://chromedriver.chromium.org/downloads

### Шаг 3: Запустить тест на одной станции

Откройте командную строку в папке проекта и выполните:

```bash
python download-one-station.py
```

Что произойдет:
1. Откроется браузер Chrome
2. Автоматически откроется страница RP5
3. Заполнятся даты
4. Нажмется кнопка "Скачать архив"
5. Подождет 15 секунд
6. Скачает файл
7. Распакует его
8. Закроет браузер

Файл сохранится в `data/rp5-csv/26063.csv`

### Шаг 4: Если тест прошел успешно - запустить массовое скачивание

```bash
python download-mass.py
```

Это скачает 5 станций:
- Санкт-Петербург (26063)
- Москва (27612)
- Ишим (28573)
- Новосибирск (29634)
- Екатеринбург (28698)

## 📝 Как изменить настройки

### Изменить станцию для теста

Откройте файл `download-one-station.py` в блокноте и измените:

```python
STATION_ID = '26063'  # Измените на нужный WMO ID
START_DATE = '01.01.2024'  # Измените дату начала
END_DATE = '31.12.2024'    # Измените дату окончания
```

### Изменить список станций для массового скачивания

Откройте файл `download-mass.py` в блокноте и измените:

```python
STATIONS = [
    '26063',  # Добавьте или удалите станции
    '27612',
    '28573',
]

START_DATE = '01.04.2016'  # Измените период
END_DATE = '01.04.2026'
```

### Скрыть браузер (работать в фоне)

В файле `download-mass.py` измените:

```python
HEADLESS = True  # Было False
```

## 🐛 Если что-то не работает

### Ошибка: "ChromeDriver не найден"

**Решение:** Убедитесь, что файл `chromedriver.exe` находится в папке `chrome-win64` в корне проекта.

### Ошибка: "selenium module not found"

**Решение:** Установите Selenium:
```bash
pip install selenium
```

### Браузер открывается, но ничего не происходит

**Решение:** Увеличьте время ожидания. Откройте `rp5_selenium_downloader.py` и измените:

```python
time.sleep(3)  # Измените на 5 или 10
```

### Кнопка "Скачать архив" не найдена

**Решение:** Возможно, сайт изменил структуру. Откройте страницу вручную и проверьте, есть ли кнопка.

### Файл не скачивается (403 Forbidden)

**Решение:** Подождите дольше после нажатия кнопки. В `rp5_selenium_downloader.py` измените:

```python
time.sleep(15)  # Измените на 30 или 60
```

## 📊 Где найти скачанные файлы

Все файлы сохраняются в папке: `data/rp5-csv/`

Формат имени: `{WMO_ID}.csv`

Например:
- `26063.csv` - Санкт-Петербург
- `27612.csv` - Москва
- `28573.csv` - Ишим

## 🎯 Быстрая проверка

### 1. Проверить, что Selenium установлен:

```bash
python -c "import selenium; print('OK')"
```

Должно вывести: `OK`

### 2. Проверить, что ChromeDriver на месте:

Откройте папку `chrome-win64` и убедитесь, что там есть `chromedriver.exe`

### 3. Запустить тест:

```bash
python download-one-station.py
```

Если все работает - увидите открывающийся браузер и процесс скачивания.

## 💡 Советы

1. **Начните с одной станции** - запустите `download-one-station.py` для теста
2. **Смотрите на браузер** - не закрывайте его вручную, пусть скрипт сам закроет
3. **Проверяйте логи** - в консоли будет показан весь процесс
4. **Не торопитесь** - между станциями есть задержка 3 секунды (это нормально)
5. **Проверяйте файлы** - после скачивания откройте CSV в Excel или блокноте
