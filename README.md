# Weather Archive - Архив погоды

Веб-приложение для просмотра архивных данных о погоде с автоматическим обновлением данных из RP5.

## 🚀 Быстрый старт

Смотрите файл [БЫСТРЫЙ-СТАРТ.md](БЫСТРЫЙ-СТАРТ.md) для подробных инструкций по запуску.

## ✨ Возможности

- Просмотр архивных данных о погоде по городам
- Автоматическое обновление данных 8 раз в день через GitHub Actions
- Поддержка множества городов по всему миру
- Визуализация погодных данных
- Статистика и анализ погоды

## 📦 Технологии

- Node.js + Express
- EJS шаблоны
- Python для загрузки данных из RP5
- GitHub Actions для автоматизации

## 🔄 Автообновление данных

Данные автоматически обновляются каждые 3 часа (8 раз в день) через GitHub Actions.
Расписание: 00:00, 03:00, 06:00, 09:00, 12:00, 15:00, 18:00, 21:00 UTC.

## 📁 Структура проекта

- `/data` - данные о городах, аэропортах, WMO кодах
- `/views` - EJS шаблоны
- `/static` - статические файлы (CSS, JS, изображения)
- `/locales` - локализация
- `/tests-archive` - архив тестовых файлов и старой документации
- `.github/workflows` - GitHub Actions для автообновления

## 🛠️ Установка и запуск

```bash
# Установка зависимостей
npm install
pip install -r requirements.txt

# Запуск сервера
node server.js
```
