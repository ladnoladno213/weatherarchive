# 📝 Шпаргалка команд

## 🚀 Первый запуск

```bash
# 1. Проверка готовности
python check-setup.py

# 2. Добавить все файлы
git add .

# 3. Закоммитить
git commit -m "Add RP5 auto-update: 8 times daily, all cities, free"

# 4. Запушить
git push
```

Затем на GitHub:
- Actions → "RP5 Frequent Updates" → Run workflow

## 🔍 Проверка результатов

```bash
# Обновить локальную копию
git pull

# Посмотреть файлы
ls data/rp5-realtime/

# Посмотреть размеры файлов
ls -lh data/rp5-realtime/ | head -20

# Посмотреть последние коммиты
git log --oneline | head -10

# Найти коммиты с обновлениями
git log --oneline | grep "Update RP5 data"
```

## 📊 Статистика

```bash
# Количество файлов
ls data/rp5-realtime/ | wc -l

# Общий размер
du -sh data/rp5-realtime/

# Последнее обновление
ls -lt data/rp5-realtime/ | head -5
```

## 🛠️ Локальный запуск (для теста)

```bash
# Установить зависимости
pip install selenium requests

# Запустить скрипт
python rp5-realtime-updater.py
```

⚠️ **Внимание:** Локальный запуск требует Chrome и ChromeDriver!

## 🔧 Настройка GitHub

```bash
# Проверить удалённый репозиторий
git remote -v

# Проверить текущую ветку
git branch

# Проверить статус
git status
```

На GitHub:
1. Settings → Actions → General
2. Workflow permissions → "Read and write permissions"
3. Save

## 📧 Мониторинг

```bash
# Посмотреть последние коммиты с датами
git log --pretty=format:"%h - %an, %ar : %s" | head -10

# Найти все обновления RP5
git log --all --grep="Update RP5 data"

# Посмотреть изменения в последнем коммите
git show HEAD --stat
```

## 📦 Создание requirements.txt

```bash

echo "requests>=2.28.0" >> requirements.txt

# Или установить и экспортировать
pip freeze | grep -E "(selenium|requests)" > requirements.txt
```

## 🔐 Проверка прав

