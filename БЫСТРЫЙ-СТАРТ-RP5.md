# Быстрый старт: Автоматическое обновление RP5

## Проблема
Тебе нужны реальные данные с RP5 каждые 3 часа, но GitHub Actions не может их скачать (403 Forbidden).

## Решение за 2 минуты

### 1. Установи зависимость
```bash
pip install schedule
```

### 2. Запусти сервис
```bash
python rp5-auto-service.py
```

### 3. Готово!
Сервис будет:
- ✅ Скачивать данные с RP5 каждые 3 часа
- ✅ Автоматически пушить в GitHub
- ✅ Render/Railway автоматически задеплоят

---

## Что дальше?

### Вариант 1: Оставить как есть
Просто держи окно открытым. Сервис работает.

### Вариант 2: Автозапуск при включении ПК
**Windows**: Task Scheduler → Create Basic Task → "When computer starts" → `python rp5-auto-service.py`

**Linux**: Создай systemd service (см. `АВТОМАТИЧЕСКОЕ-ОБНОВЛЕНИЕ-RP5.md`)

### Вариант 3: Запустить на VPS
Если не хочешь держать компьютер включенным:
- DigitalOcean/Linode/Vultr - $5/месяц
- AWS/Google Cloud Free Tier - бесплатно
