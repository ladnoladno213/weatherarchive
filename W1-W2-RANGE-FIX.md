# Исправление логики W1/W2 - проверка диапазона часов

## Проблема

Предыдущая реализация смотрела только на ОДИН конкретный час:
- W1 смотрел только на час i-3
- W2 смотрел только на час i-6

Это приводило к тому, что:
1. Если в час i-3 не было явления, но в i-4, i-5, i-6 было - W1 оставался пустым
2. Если в час i-6 не было явления, но в i-7, i-8, ..., i-12 было - W2 оставался пустым
3. Грозы "терялись", если они были не точно в час i-3 или i-6

## Пример из скриншота (5-6 июля 2024, Ишим)

**5 июля 23:00**:
- Было: W1 пустой, W2="Гроза (грозы) с осадками или без них."
- Проблема: Гроза попала в W2 вместо W1

**6 июля 05:00**:
- Было: W1 пустой, W2="Ливень (ливни)."
- Проблема: Ливень попал в W2 вместо W1

**6 июля 20:00**:
- Было: WW="Ливневый(ые) дождь(и) слабый(ые)...", W1 пустой, W2 пустой
- Проблема: Ливень в WW, но W1/W2 пустые (должен быть "Ливень (ливни)." в W1)

## Решение

Теперь логика проверяет ДИАПАЗОН часов:

### W1 (прошедшая погода за 3-6 часов)
Проверяем часы: i-3, i-4, i-5, i-6

```javascript
for (let j = 3; j <= 6 && i >= j; j++) {
  const prevCode = h.weathercode?.[i - j] ?? null;
  // Проверяем каждый час в диапазоне
  // Выбираем явление с наивысшим приоритетом
}
```

### W2 (прошедшая погода за 6-12 часов)
Проверяем часы: i-6, i-7, i-8, i-9, i-10, i-11, i-12

```javascript
for (let j = 6; j <= 12 && i >= j; j++) {
  const prevCode = h.weathercode?.[i - j] ?? null;
  // Проверяем каждый час в диапазоне
  // Выбираем явление с наивысшим приоритетом
}
```

### Приоритеты явлений

```javascript
const w1Priorities = {
  9: 100, // Гроза (самый высокий приоритет)
  8: 90,  // Ливень
  7: 80,  // Снег
  6: 70,  // Дождь
  5: 60,  // Морось
  4: 50,  // Туман
  3: 40,  // Метель
};
```

Если в диапазоне было несколько явлений, выбирается самое важное (с наивысшим приоритетом).

## Логика гроз

Грозы также проверяются по диапазонам:

### Диапазон W1 (3-6 часов)
```javascript
for (let j = 3; j <= 6 && i >= j; j++) {
  const prevAnalysis = analyzeThunderstorm(multiModelData, times[i - j], prevCl);
  if (prevAnalysis.hasThunderstorm) {
    hadThunderstormInW1Range = true;
    break;
  }
}
```

### Диапазон W2 (6-12 часов)
```javascript
for (let j = 6; j <= 12 && i >= j; j++) {
  const prevAnalysis = analyzeThunderstorm(multiModelData, times[i - j], prevCl);
  if (prevAnalysis.hasThunderstorm) {
    hadThunderstormInW2Range = true;
    break;
  }
}
```

## Правила заполнения

### 1. Текущая гроза
```
WW: "Гроза слабая или умеренная..."
W1: "Гроза (грозы) с осадками или без них."
W2: "Ливень (ливни)." (если были ливни в диапазоне 6-12 часов)
```

### 2. Гроза в предыдущем часе
```
WW: "Гроза в течение последнего часа. Ливневый дождь." (если есть текущее явление)
    или "Состояние неба в общем не изменилось." (если нет текущего явления)
W1: "Гроза (грозы) с осадками или без них."
W2: зависит от того, что было в диапазоне 6-12 часов
```

### 3. Гроза в диапазоне W1 (3-6 часов)
```
WW: текущее явление (или пусто)
W1: "Гроза (грозы) с осадками или без них."
W2: зависит от того, что было в диапазоне 6-12 часов
```

### 4. Гроза в диапазоне W2 (6-12 часов)
```
WW: текущее явление (или пусто)
W1: зависит от того, что было в диапазоне 3-6 часов
W2: "Гроза (грозы) с осадками или без них."
```

### 5. Ливень без грозы
```
WW: "Ливневый(ые) дождь(и) слабый(ые)..."
W1: "Ливень (ливни)." (если был ливень в диапазоне 3-6 часов)
W2: "Облака покрывали более половины неба..." (если в W1 есть явление)
    или "Ливень (ливни)." (если в W1 нет явления, но был ливень в диапазоне 6-12 часов)
```

## Правило "не дублировать"

Если W1 и W2 одинаковые, убираем W2:

```javascript
if (W1 && W2 && W1 === W2) {
  W2 = null;
}
```

Пример:
```
W1: "Ливень (ливни)."
W2: "Ливень (ливни)."
→ W2 становится null
```

## Правило для W2 (облачность)

Если в W1 есть реальное явление, в W2 можно показать облачность:

```javascript
if (W1) {
  // В W1 есть явление - можем показать облачность в W2
  W2 = w2Text; // может быть облачность
} else {
  // В W1 нет явления - показываем только реальное явление в W2
  if (w2Text && !w2Text.includes('Облака покрывали')) {
    W2 = w2Text;
  }
}
```
