# Как обновлять архивы RP5

## Простой способ (раз в месяц, 5 минут)

### Шаг 1: Запусти скрипт

```bash
python rp5-monthly-update.py
```

### Шаг 2: Жди

Скрипт:
1. Откроет браузер
2. Зайдет на RP5
3. Скачает архивы для всех станций
4. Спросит, хочешь ли закоммитить в GitHub

### Шаг 3: Подтверди

Когда скрипт спросит "Хочешь закоммитить и запушить в GitHub? (y/n)", напиши `y` и нажми Enter.

### Готово!

Данные автоматически загрузятся в GitHub, и Render/Railway задеплоят обновленную версию.

---

## Что делает скрипт?

1. ✅ Открывает RP5 для каждой станции
2. ✅ Заполняет формы (даты, формат, кодировка)
3. ✅ Скачивает архивы
4. ✅ Распаковывает .gz файлы
5. ✅ Сохраняет в `data/rp5-csv/`
6. ✅ Коммитит и пушит в GitHub

1. Открой https://rp5.ru/
2. Найди нужный город
3. В URL будет `wmo_id=XXXXX` - это и есть WMO ID

## Частые вопросы

### Как часто нужно обновлять?
Раз в месяц достаточно. Архивные данные не меняются часто.

### Можно ли автоматизировать?
Да, можно настроить Task Scheduler (Windows) или cron (Linux/Mac) для автоматического запуска скрипта.

**Windows (Task Scheduler)**:
1. Открой Task Scheduler
2. Create Basic Task
3. Trigger: Monthly (1-го числа)
4. Action: Start a program
5. Program: `python`
6. Arguments: `C:\path\to\rp5-monthly-update.py`

**Linux/Mac (cron)**:
```bash
# Открой crontab
crontab -e

# Добавь строку (запуск 1-го числа каждого месяца в 3:00)
0 3 1 * * cd /path/to/project && python3 rp5-monthly-update.py
```

### Что если скрипт упал?
Просто запусти еще раз. Скрипт безопасен и не сломает данные.

### Нужен ли интернет?
Да, скрипт скачивает данные с RP5 и пушит в GitHub.

### Можно ли запускать на сервере?
Нет, скрипт требует браузер (Selenium). Запускай на своем компьютере.

## Альтернатива: Ручное скачивание

Если не хочешь использовать скрипт:

1. Открой https://rp5.ru/archive.php?wmo_id=28573&lang=ru
2. Вкладка "Скачать архив"
3. Выбери период, CSV, UTF-8
4. Нажми "Выбрать в файл"
5. Скачай и распакуй .gz
6. Положи .csv в `data/rp5-csv/`
7. Закоммить:
   ```bash
   git add data/rp5-csv/*.csv
   git commit -m "Update RP5 archives"
   git push origin main
   ```
