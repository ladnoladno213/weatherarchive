# Паттерны RP5 за 2024 год и рекомендации по улучшению

## Ключевые находки из анализа 2923 записей

### 1. МОРОСЬ - ОЧЕНЬ РЕДКОЕ ЯВЛЕНИЕ

**Всего случаев мороси за 2024 год: 11 (0.4%)**

Это подтверждает, что Open-Meteo действительно путает морось и ливни. В реальных данных RP5:
- Морось встречается крайне редко (11 раз за год)
- Ливни встречаются часто (142 раза за год)
- Соотношение: 1 морось на 13 ливней

**ВЫВОД**: Решение заменять морось на ливень (коды 50-57 → 80) ПРАВИЛЬНОЕ и соответствует реальности.

### 2. ПРАВИЛО ДЛЯ W2 ПОДТВЕРЖДЕНО

Анализ показывает четкий паттерн:

**Когда W2 показывает облачность (454 случая):**
- W1 = "Ливень (ливни)." - 346 раз
- W1 = "Снег и/или другие виды твердых осадков" - 227 раз
- W1 = "Метель" - 124 раз
- W1 = "Гроза (грозы) с осадками или без них." - 73 раза

**Когда W2 показывает явление (не облачность):**
- W2 = "Метель" - 123 раза
- W2 = "Ливень (ливни)." - 75 раз
- W2 = "Снег и/или другие виды твердых осадков" - 15 раз

**ПРАВИЛО ПОДТВЕРЖДЕНО**: 
- Если в W1 есть реальное явление → W2 показывает облачность
- Если в W1 нет явления → W2 показывает реальное явление

### 3. САМЫЕ ЧАСТЫЕ КОМБИНАЦИИ

#### Топ-5 комбинаций WW/W1/W2:

1. **1933 раза**: WW пустой, W1 пустой, W2 пустой
   - Это ясная погода без явлений
   - Правильно, что ничего не показываем

2. **137 раз**: WW="Состояние неба в общем не изменилось", W1="Ливень (ливни).", W2="Облака покрывали более половины неба..."
   - Сейчас ничего не происходит, но 3-6 часов назад был ливень
   - W2 показывает облачность, потому что в W1 есть явление

3. **66 раз**: WW="Ливневый дождь слабый", W1="Ливень (ливни).", W2="Облака покрывали более половины неба..."
   - Сейчас ливень, 3-6 часов назад тоже был ливень
   - W2 показывает облачность

4. **59 раз**: WW="Снег непрерывный слабый", W1="Снег и/или другие виды твердых осадков", W2="Облака покрывали более половины неба..."
   - Сейчас снег, 3-6 часов назад тоже был снег
   - W2 показывает облачность

5. **54 раза**: WW="Снег непрерывный слабый", W1="Снег и/или другие виды твердых осадков", W2="Метель"
   - Сейчас снег, 3-6 часов назад был снег, 6-12 часов назад была метель
   - W2 показывает явление (метель), потому что это важнее облачности

### 4. ГРОЗЫ - СПЕЦИАЛЬНЫЙ ПАТТЕРН

**Всего гроз за 2024 год: 27 случаев**

Типы гроз:
- "Гроза слабая или умеренная без града, но с дождем и/или снегом" - 10 раз
- "Гроза, но без осадков" - 9 раз
- "Гроза (с осадками или без них)" - 6 раз
- "Видна молния, грома не слышно" (зарница) - 2 раза

**Паттерн для гроз:**
- WW: "Гроза слабая или умеренная..."
- W1: "Гроза (грозы) с осадками или без них."
- W2: "Ливень (ливни)."

Это показывает, что:
1. Грозе обычно предшествуют ливни (W2)
2. После грозы часто пишут "Состояние неба в общем не изменилось" в WW
3. W1 показывает "Гроза", W2 показывает "Ливень"

### 5. "СОСТОЯНИЕ НЕБА В ОБЩЕМ НЕ ИЗМЕНИЛОСЬ" - 353 РАЗА

Это второе по частоте значение WW после пустого.

Используется когда:
- Нет текущих явлений
- Но были явления в прошлом (W1/W2 заполнены)
- Облачность стабильная

Примеры:
- WW="Состояние неба...", W1="Ливень (ливни).", W2="Облака покрывали более половины неба..." - 137 раз
- WW="Состояние неба...", W1="Гроза (грозы)...", W2="Ливень (ливни)." - 38 раз
- WW="Состояние неба...", W1="Снег и/или другие виды твердых осадков", W2="Облака покрывали более половины неба..." - 25 раз

## РЕКОМЕНДАЦИИ ПО УЛУЧШЕНИЮ

### 1. Морось → Ливень (УЖЕ РЕАЛИЗОВАНО)

```javascript
// Корректируем weathercode: заменяем морось на ливень
if (wcode >= 50 && wcode <= 57) {
  correctedWcode = 80; // Ливневый дождь слабый
}
```

**СТАТУС**: ✅ Правильно реализовано

### 2. Правило для W2 (УЖЕ РЕАЛИЗОВАНО)

```javascript
// Правило из RP5: если в W1 есть реальное явление, то в W2 можно показать облачность
if (W1) {
  // В W1 есть явление - можем показать облачность в W2
  W2 = w2Text;
} else {
  // В W1 нет явления - показываем только реальное явление в W2
  if (w2Text && !w2Text.includes('Облака покрывали')) {
    W2 = w2Text;
  }
}
```

**СТАТУС**: ✅ Правильно реализовано

### 3. НОВОЕ: Добавить "Состояние неба в общем не изменилось"

Когда использовать:
- WW пустой (нет текущих явлений)
- W1 или W2 заполнены (были явления в прошлом)
- Облачность > 50%

```javascript
// Если WW пустой, но были явления в прошлом, показываем "Состояние неба в общем не изменилось"
if (!WW && (W1 || W2) && Cl >= 50) {
  WW = 'Состояние неба в общем не изменилось.';
}
```

**СТАТУС**: ⚠️ НУЖНО ДОБАВИТЬ

### 4. НОВОЕ: Улучшить логику гроз

Текущая логика гроз работает, но можно улучшить:

```javascript
// После грозы (в предыдущем часе), если сейчас нет явлений
if (hadThunderstormLastHour && !analysis.hasThunderstorm && !WW) {
  WW = 'Состояние неба в общем не изменилось.';
  W1 = 'Гроза (грозы) с осадками или без них.';
  // W2 остается как есть (обычно "Ливень (ливни).")
}
```

**СТАТУС**: ⚠️ НУЖНО УЛУЧШИТЬ

### 5. НОВОЕ: Дедупликация WW

Текущая дедупликация работает, но можно улучшить:

```javascript
// Не показываем "Состояние неба в общем не изменилось" более 3 раз подряд
if (WW === 'Состояние неба в общем не изменилось.' && lastWW === WW) {
  unchangedSkyCount++;
  if (unchangedSkyCount > 3) {
    WW = null; // Скрываем после 3 повторений
  }
} else {
  unchangedSkyCount = 1;
}
```

**СТАТУС**: ⚠️ НУЖНО ДОБАВИТЬ

### 6. НОВОЕ: Приоритет явлений

Из анализа видно, что приоритет явлений:
1. Гроза (самый высокий)
2. Ливень
